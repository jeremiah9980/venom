# Player Photos — Consent-Gated Slots

Drop a parent-cleared photo here named after the player slug, and it will automatically appear on the roster card. If no file is present, the card shows the styled jersey-number fallback instead. Same behavior for every player — no HTML edits required.

## Filename convention
`<firstname>-<lastinitial>.jpg` (lowercase, hyphenated)

## Before adding a photo
1. Written media release on file from the player's parent/guardian, scoped to public web display on the team site.
2. The photo itself was provided or approved by the family (not pulled from third-party social).
3. If the family later asks for removal, delete the file. The card reverts to the jersey-number fallback on the next page load — no other edits needed.
